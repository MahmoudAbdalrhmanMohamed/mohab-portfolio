# Mohab Mahmoud Portfolio

Native static portfolio built with HTML, CSS, and JavaScript. No Nuxt, no build step.

## How to use
Open `index.html` directly in a browser, or upload the folder to any static host.

## Edit before publishing
- Replace `https://example.com/` in `index.html`, `public/sitemap.xml`, and `public/robots.txt` with your real domain.
- Add LinkedIn/GitHub links if needed.
- Update `assets/profile-data.js` when the CV content changes.
- Replace `assets/profile.webp` to change the hero image.
